Selecting a Device

From the top menu bar, execute Device > Select or press F2.

The Device Selection dialog box pops up. Select a device from the devices list box and click OK.

As soon as the device is selected, the Device parameters area is updated and lets you see the selected device special bytes. The Buffer information area is updated as well with device dependent information.

Each device supports a set of available communication media (RS232, USB or CAN).
