Selecting an ISP Hardware

TYPE

SWITCH

OPTIONAL

ARGUMENTS

Setup

-hardware

Yes

RS232 | USB | ATMEL_RS232 | VECTOR | IXXAT | PEAK | SYSTECUSBCAN |  USB_LPC | USB_UART | USB_SPI

The -hardware switch is used to select the communication medium.

Example of use :

> batchisp -device t89c51cc01 -hardware IXXAT -operation erase F
