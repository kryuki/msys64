Setting the AutoISP Mode

TYPE

SWITCH

OPTIONAL

ARGUMENTS

Setup

-autoisp

Yes

{ 0 | 1 } { 0 | 1 }

Example of use :

> batchisp -device at89c51rd2 -autoisp -operation erase F -demo

The -autoisp switch may have two arguments or no argument at all; there is no other possible number of arguments.

The first argument is the RESET signal active level. The second argument is the PSEN signal active level.
