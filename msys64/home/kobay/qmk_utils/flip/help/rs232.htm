Trouble Shooting RS232

Please read first the Setting Up the RS232 Interface topic.

 

Although the Targus PA088 USB to RS232 adapter is supported by Flip and Batchisp, it requires a specific Atmel's DLL (AtRs232Targus.dll) to work properly. If you work with a Targus adapter, please proceed as follows:

  • exit the Flip program if it was running

  • in the \bin folder of the Flip package, rename AtRs232.dll to AtRs232_old.dll

  • rename AtRs232Targus.dll to AtRs232.dll

  • start the Flip or the Batchisp program
