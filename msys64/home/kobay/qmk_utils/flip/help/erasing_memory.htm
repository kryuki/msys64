Erasing Memory

Execute Device > Erase...

 

In the Erase dialog, select the Full Chip Erase or the Erase blocks option. Check the blocks you want to be erased if the Erase blocks option is active.

Performing a full chip erase usually resets the device special bytes : the Software Security Byte, the Hardware Byte, the Boot Status Byte and the Software Boot Vector are reset to their default values.
