USB Driver Installation

Set the ISP mode hardware conditions and reset the target hardware. The New Hardware Detection Wizard pops up.

 

Step 1 -  On USB device connection, the first Wizard dialog pops up; select the No, not this time option, then click Next.

 

Step 2 - On the second dialog, select the Install from a list or specific location option, then click Next.

 

Step 3 - Set the Driver search options to Search for the best driver in these locations; check the Include this location in the search and enter the /usb folder pathname of the Flip package. Click Next.

 

Step 4 - The Wizard installs the new hardware driver and displays the following dialog. Click Finish and you are done.

 

Step 5 - You may open the Device Manager window to check the proper installation of the new device.

 

Please read Trouble Shooting USB in case of USB device malfunction.
