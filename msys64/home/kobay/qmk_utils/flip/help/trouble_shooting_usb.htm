Trouble Shooting USB

It may happen that the USB device is not properly recognized by the Operating System. In such a situation, Flip pops up a Cannot open port message when you click the Open button of the USB Port Connection dialog.

You may confirm a device driver problem by opening the Peripheral Manager. The failing device is usually marked with a yellow circle containing an exclamation mark.

Did you set the USB device in ISP conditions?

In normal operation, the microcontroller starts executing your firmware on reset. Entering In-System Programming mode requires special hardware conditions to be met. Before attempting to open the USB port from Flip, you have to force the target USB device to enumerate for ISP by setting the ISP hardware conditions. Did you already install Flip on this machine?

If you are sure that the USB device enumerates for ISP but remains yellow-marked in the Device Manager window, read the following.

Flip versions prior to 3.1.0 had a different USB driver installation procedure. We recommend that you delete any previous driver registration and erase all INF files installed by previous versions of Flip. This procedure is described in the Readme file of the /info folder/.
