Overview

The Atmel's ISP functions library (named AtJniIsp) is released as a DLL so that you can write your own In-System Programming application without knowing the details of our ISP protocols. FLIP and batchisp programs themselves rely on this library.

The ISP functions prototypes can be found in the AtJniIsp.h file.

A demo program is provided, with C-source code as a quick start guide.

Although this DLL has been built using Microsoft Visual C++, it can be linked with an executable file built using any vendor's tools.
