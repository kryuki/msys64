Selecting the Target Memory

Some microcontrollers house a Flash memory and an EEPROM one. FLIP lets you program both.

When multiple target memories exist and if the bootloader allows programming them, a memory selection button is displayed in the bottom center of the main window. By default, the Flash memory is selected; clicking the button toggles the selection to the next memory (usually the EEPROM). If you click the button, its color and label change as well as the Operations Flow frame title color, the Memory Buffer Information frame title color and the buffer edition window color.
